# 📘 Study: Building Testable APIs with Node.js

## 📌 About the Project
This project is a comprehensive technical study based on the book **"Building Testable APIs with Node.js"**. 

Using real-world scenarios, this repository explores the creation of **production-grade Node.js applications** by following industry best practices for quality and software development. It serves as a guide for building robust, resilient, and **test-driven systems**, covering everything from architectural foundations to advanced security and testing strategies.

---

## 📖 Source Material
This study is based on the open-source book available online. You can access it here:

> **Language Note:** The source book and its original content are written in **Portuguese (PT-BR)**, though this repository's documentation and code comments are maintained in English.

## 🚀 Tech Stack

### **Backend & Database**
* **Runtime:** Node.js
* **Framework:** Express
* **Database:** MongoDB with Mongoose (ODM)
* **Security:** JWT (JSON Web Tokens), Bcrypt (Hashing), and ACL (Access Control List)

### **Testing & Quality**
* **Test Runner:** Mocha
* **Assertion Library:** Chai
* **Mocks/Spies/Stubs:** Sinon.js
* **Integration Testing:** Supertest
* **Linting:** ESLint & Prettier

### **Architecture & Patterns**
* **Design Pattern:** Singleton
* **Software Pattern:** MVC (Model-View-Controller)
* **Development Flow:** TDD & BDD
